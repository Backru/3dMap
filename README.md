# Vue3 可视化大屏项目

一个基于Vue3、Three.js和ECharts的数据可视化大屏项目。

## 功能特性

### 🗺️ 3D中国地图 (专业级特效版)
- **真实GeoJSON数据**: 集成阿里云DataV真实省份边界数据
- **Shader渐变特效**: 自定义Shader材质，支持时间动画和高度渐变
- **CSS3D标签系统**: 省份标签、数据卡片、散点图标签
- **镜像倒影效果**: 地面镜面反射，增强立体感
- **飞线动画系统**: 城市间动态飞线，支持贝塞尔曲线路径
- **粒子特效**: 网格粒子扩散、上升粒子、点状粒子系统
- **柱状图特效**: Shader渐变柱状图，支持GSAP动画
- **GSAP进场动画**: 省份、柱状图、散点图的弹性进场效果
- **动态交互效果**: 
  - 省份悬停高亮和缩放
  - 城市点击凸起动画
  - 平滑的相机过渡动画
- **视觉增强**: 
  - 发光边框和材质
  - 浮动动画效果
  - 数据驱动的颜色和高度
- **信息展示**: 实时显示悬停省份的详细信息

### 📊 多样化图表
- **销售趋势图**: 折线图展示月度销售数据
- **产品分类占比**: 饼图展示各类产品占比
- **流量来源分析**: 横向柱状图展示流量来源
- **实时访问统计**: 双轴图展示PV/UV数据

### 🎨 视觉效果
- 科技感蓝色主题
- 渐变背景和边框
- 动态光效和动画
- 响应式布局设计

## 安装和运行

1. 安装依赖:
```bash
npm install
```

2. 启动开发服务器:
```bash
npm run dev
```

3. 构建生产版本:
```bash
npm run build
```

## 技术栈

- **Vue 3**: 前端框架
- **Three.js**: 3D图形库，用于地图渲染
- **GSAP**: 高性能动画库
- **CSS3DRenderer**: Three.js CSS3D渲染器
- **WebGL Shaders**: 自定义顶点和片段着色器
- **GeoJSON**: 真实地理数据格式
- **ECharts**: 图表库，用于数据可视化
- **Vite**: 构建工具

## 核心特效技术

### 🎨 Shader特效
- **地图Shader**: 高度渐变、边缘发光、时间动画
- **柱状图Shader**: 垂直渐变、发光效果
- **边缘特效Shader**: 上升光带效果
- **粒子Shader**: 粒子生命周期、速度控制

### 🌟 动画系统
- **GSAP Timeline**: 复杂动画序列
- **Three.js Tween**: 相机和物体过渡
- **Shader动画**: GPU加速的材质动画
- **CSS3D动画**: 标签和UI元素动画

## 交互说明

### 地图交互 (全新体验)
- **省份交互**:
  - 鼠标悬停: 省份高亮、缩放和信息显示
  - 点击省份: 平滑切换到省级视图
  - 下拉选择: 快速跳转到指定省份
- **城市交互**:
  - 悬停效果: 城市发光和缩放
  - 点击动画: 城市凸起的弹跳效果
  - 数据可视化: 城市高度反映数据大小
- **视图切换**:
  - 相机动画: 平滑的3D视角过渡
  - 省份聚焦: 选中省份居中放大显示
  - 返回全国: 选择"全国"返回整体视图

### 图表特性
- 所有图表都有自动轮播提示效果
- 实时数据图表会定期更新数据
- 支持鼠标悬停查看详细数据
- 响应式设计，适配不同屏幕尺寸

## 数据说明

项目使用模拟数据，包括:
- 31个省份的经济数据
- 12个月的销售趋势数据
- 6个产品分类的占比数据
- 5个流量来源的统计数据
- 24小时的实时访问数据

## 自定义开发

### 添加新图表
1. 在 `src/components/` 目录创建新的Vue组件
2. 在 `src/data/mockData.js` 中添加对应的数据
3. 在 `App.vue` 中引入并使用新组件

### 修改地图数据
1. 编辑 `src/components/ChinaMap.vue` 中的 `provinceGeometry` 对象
2. 更新 `src/data/mockData.js` 中的省份数据

### 样式定制
1. 修改 `src/style.css` 中的CSS变量和样式
2. 调整各组件中的ECharts配置项
