## ⚙️ 高级配置

### 调整模型大小

在 `setupLoadedModel` 函数中：

```javascript
// 修改这个值来调整大小
const scale = 8 / maxDim  // 8 是目标大小，可以改成 10、12 等
```

### 调整模型位置

```javascript
model.position.y = -box.min.y * scale + 0.5  // +0.5 让车悬空一点
```

### 识别车身材质

代码会自动识别包含这些关键词的材质：
- "paint"
- "body"
- "car"

或者颜色饱和度 > 0.3 的材质

## ❓ 常见问题

### Q: 模型不显示？
**A:** 
1. 打开浏览器控制台（F12）查看错误
2. 确认文件路径正确
3. 确认文件格式是 `.glb` 或 `.gltf`
4. 检查文件是否损坏

### Q: 模型太大或太小？
**A:** 
代码会自动缩放，但如果还是不合适：
- 修改 `setupLoadedModel` 中的 `scale` 计算
- 或在 Blender 中调整模型尺寸后重新导出

### Q: 模型没有颜色？
**A:** 
- 确保模型包含材质
- 检查材质名称是否包含 "paint"、"body" 等关键词
- 或手动修改 `changeColor` 函数中的识别逻辑

### Q: 加载很慢？
**A:** 
- 使用 GLB 格式（已压缩）
- 使用在线工具压缩模型
- 考虑使用低面数模型
- 使用 Draco 压缩（需要特殊导出）

### Q: 车门不能打开？
**A:** 
加载的模型需要有正确的骨骼/动画才能打开车门。
大多数免费模型是静态的，不支持车门动画。
